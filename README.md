## Getting started

**Installing**
> Cloning the repository

```bash
$ git clone https://github.com/biagavirete/superhero_guide.git
$ cd superhero_guide
```

**Running**
> Installing dependencies

```bash
$ yarn
```

> Running React

```bash
$  yarn start
```

## Built with

* Javascript
* React
* React-router-dom
* React Helmet
